## 使用指南

### 何时使用此模板

- 引入新技术栈或框架
- 架构方案选型
- 重大技术决策
- 性能优化方案评估
- 安全方案选型

### 填写建议

1. **背景部分**：确保问题陈述清晰，避免模糊表述
2. **方案对比**：使用量化指标，避免主观判断
3. **POC 验证**：必须包含明确的验证目标和通过标准
4. **结论建议**：推荐方案必须有数据支撑，避免"感觉更好"

### 审核要点

- [ ] 调研目标是否明确且可验证
- [ ] 候选方案是否覆盖主要选项
- [ ] 对比维度是否全面
- [ ] 评分权重是否合理
- [ ] 参考资料是否可追溯
- [ ] 结论是否有数据支撑
